# gstack-agent Antigravity adapter

This branch is specialized for the Antigravity runtime. It includes the shared workflow skills from main and adds Antigravity-specific adapter configuration.

Use git checkout antigravity to select this runtime target and run python scripts/validate_branches.py to validate the branch artifacts.
