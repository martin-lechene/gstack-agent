# gstack-agent Codex adapter

This branch is specialized for the OpenAI Codex runtime. It includes the shared workflow skills from main and adds Codex-specific adapter configuration.

Use git checkout codex to select this runtime target and run python scripts/validate_branches.py to validate the branch artifacts.

Adapter files:
- CODEX.md
- codex.json

The branch is kept in sync with main while tailoring configuration to Codex execution.
