# gstack-agent VS Code AI adapter

This branch is specialized for the VS Code AI runtime. It includes the shared workflow skills from main and adds VS Code-specific adapter configuration.

Use git checkout vscode to select this runtime target and run python scripts/validate_branches.py to validate the branch artifacts.

Adapter files:
- VSCODE.md
- .vscode/agent.json

The branch is kept in sync with main while tailoring configuration to VS Code AI execution.
