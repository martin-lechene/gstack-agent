# gstack-agent PhpStorm Junie adapter

This branch is specialized for the PhpStorm Junie runtime. It includes the shared workflow skills from main and adds PhpStorm-specific adapter configuration.

Use git checkout phpstorm to select this runtime target and run python scripts/validate_branches.py to validate the branch artifacts.

Adapter files:
- PHPSTORM.md
- .phpstorm/junie.json

The branch is kept in sync with main while tailoring configuration to PhpStorm Junie execution.
