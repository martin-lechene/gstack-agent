# gstack-agent Copilot adapter

This branch is specialized for the GitHub Copilot runtime. It includes the shared workflow skills from main and adds Copilot-specific adapter configuration.

Use git checkout copilot to select this runtime target and run python scripts/validate_branches.py to validate the branch artifacts.

Adapter files:
- COPILOT.md
- copilot.json

The branch is kept in sync with main while tailoring configuration to GitHub Copilot execution.
